2) Installation - top

Please follow these steps to setup Plugins For Wowonder:

  • Installation is simple but make sure you have update Wowonder to version 1.4 Upload the files from the "Script" folder on the FTP server (usually public_html folder when installing it on a domain).

    1. Open you cpanel
    2. In box Files find filemanage
    3. Select Web Root
    4. Search in new windows "Update"
    5. Select File "Script/" in a zip comprissed select and upload
    6. Extract code in Public_html/
    7. Open in you website install/install_ads.php and press "install/update"
    8. Active or disable plugins in Admin plugins Example URL: http://wo.mired10.com/index.php?link1=admin-plugins
    9. Configure.... And ready!!
